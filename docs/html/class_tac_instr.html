Detailed Description

A generic three-address code instruction. It will store:

  • the instruction's valuenumber
  • the operator
    See also
    oprEnum

Constructor of a 3-address code instruction. The result is internally stored as an InstrAddress representing the value number (e.g corresponding to an index to the code array)

Parameters
opThe operator for this instruction, as an oprEnum
operand1The first operand, as a generic Address
operand2The second operand (may be NULL for operators that do not require 2 operands)
operand3Holds a temporary, when explicitly needed to specify the address result, depending on the operation
